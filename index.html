Picked.Email

PickedMail brings true individualization to email by picking out the emails you care about. It starts with a generic classification, but once you start using it, it quickly adopt your personal preference.

PickedMail also allows you to create any custom tag you want, and auto tag incoming emails. You can choose the set of tags to get push notification for.
